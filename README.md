## Employee Portal using NextJS
- Made by: Michelle - c14240169
- Deployed using Vercel: [employee-portal-next-pbx8.vercel.app](https://employee-portal-next-pbx8.vercel.app/)
- Create an account first to access the dashboard. Make sure to confirm your email before logging-in using your email account you just created.
- Node package using `pnpm`
- For route protection, it's configured under `@lib/supabase/proxy.ts` and for the error handling (error message) is configured under `@app/actions/auth.ts` using ActionState.
